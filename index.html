Projects

Brainiac

2014

An arcade-style game designed to help international students learn aspects of English, Math, Chemistry, and as a bonus - names of peculiar programming languages. It was 'virally' uploaded onto several computers in the school lab and thoroughly tested with the help and advice of subject experts (teachers).
Brainiac was my first ever project, and it was completed in collaboration with a friend in two weeks as a final project for a highschool course. Later on, it was rewritten and refactored using Python.

Python, prev. SmallBasic

Wandaless

2016

A project completed and presented for HackHarvard 2016. The web application lets users collaborate on travel plans. Users can connect and chat real-time, set a starting point to meet up, change destination nodes which they want to visit during their trip and visualize the entire route on a map.
Additionally, Wanda, the responsive chat bot, analyzes chat messages to sense when users are suggesting additional destination nodes or asking for a trip review and responds to the requests accordingly.
Wandaless was completed in a group of four, with my specialization being real-time chat interfaces and AI.

Javascript (incl. Ajax, jQuery), Java; Bootstrap, Google Cloud Services, Googlemaps API and Motion.ai
