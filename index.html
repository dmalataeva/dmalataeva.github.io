PROJECTS

Visualyrics

2017

Visualyrics is a simple game based on a machine's emotional perception of songs. The program searches for a song's lyrics and uses machine learning to analyze its words in terms of sentiment. Then the program outputs this data as a user-friendly audio visualizer, and it is up to the user to recognize the song.

Built using:

  • Python
  • Javascript (WebGL)
  • Watson (IBM) API
  • Semantria (Lexalytics) API
  • Genius API
  • HTML & CSS

Resourceful

2016

Resourceful's main purpose is to reduce food waste that is caused by unused raw foods and ingredients. The program, integrated as an Amazon Echo skill, helps people find the perfect recipe based on the ingredients they already have at home. Resourceful is also able to accomodate special needs like allergies and dietary restrictions.

Built using:

  • Java
  • Amazon Echo
  • AWS
  • Edamam API

Brainiac

2014

An arcade-style game designed to help international students learn topics in English, Math, Chemistry, and Computer Science. It was 'virally' uploaded onto several computers in a highschool lab and thoroughly tested with the help and advice of subject experts (teachers).

Built using:

  • Python
  • SmallBasic
  • Photoshop & Illustrator

Wandaless

2016

Wandaless is a webapp that allows groups of people to collaborate on travel plans.
Users can connect, chat real-time and democratically plan out their trip - all the way from starting point to finish.

Built using:

  • Java
  • Javascript
  • Google Cloud Services
  • GoogleMaps API
  • Motion.ai (chatbot)

Transactual

2016

Transactual is designed to provide people with day-to-day payment information with minimized software installation and configuration. A background app is triggered by the NFC activity of your phone - it checks if payments are being made, scrapes information from receipts on your screen and stores it into an account associated with your phone number. Simply text Transactual to receive a list of your payments for various time periods.

Built using:

  • Java for Android
  • ocr.space
  • PHP & MySQL
  • HTML & CSS
